## macOS Shortcuts

| macOS Shortcuts | Description         |
| --------------- | ------------------- |
| ⇧ ⌘  . ( dot )  | Show hidden folders |
| ⌘  ,（ comma ） | Preference          |
| ⌥  ⌘  8         | Scale up or down    |
| ⌥  ⌘  -         | Scale down          |
| ⌥  ⌘  +         | Scale up            |
| ⇧  ⌘  /         | Show help           |
| ⌘ Space         | Spotlight Search    |
| ⌥  ⌘ Space      | Finder Search       |
| ⇧  ⌘  5         | Screenshot          |
| ⌃ 1/2/3         | Switch desktop      |
| ⌥  ⌘ D          | Show Dock           |
| ⌥  ⌘ S          | Show the Launchpad  |
| ⌃ ⌘ Q           | Lock screen         |
