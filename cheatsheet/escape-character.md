| Escape character | Meaning                         | ASCII |
| ---------------- | ------------------------------- | ----- |
| `\0`             | NUL (null)                      | 000   |
| `\a`             | BEL (bell)                      | 007   |
| `\b`             | BS ( backspace )                | 008   |
| `\t`             | HT (  horizontal vertival tab ) | 009   |
| `\n`             | LF ( line feed )                | 010   |
| `\v`             | VT (  vertival tab )            | 011   |
| `\f`             | FF （ form feed ）              | 012   |
| `\r`             | CR ( carriage return )          | 013   |
| `\"`             | double quotation mark           | 034   |
| `\'`             | single quotation mark           | 039   |
| `\?`             | question mark `?`               | 063   |
| `\\`             | backslash`\`                    | 092   |
